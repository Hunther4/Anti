# High-Density Output Protocol

## Objetivo
Generar texto con máximo valor semántico por token, evitando filler words como 'es decir', 'basado en', etc.

## Pasos para Destilar Info
1. **Identificar núcleos**: Extraer solo términos clave que portan significado (sustantivos técnicos, verbos de acción, números precisos).
2. **Eliminar redundancia**: Remover repeticiones y explicaciones innecesarias - usar definiciones implícitas cuando sea posible.
3. **Consolidar conceptos**: Unir ideas relacionadas en una sola oración para mantener flujo continuo.

## Anti-patrón
No copiar fragmentos de fuentes sin procesarlos; siempre sintetizar la información original.
