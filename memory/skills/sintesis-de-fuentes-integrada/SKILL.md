## Objetivo
Maximizar la densidad informativa al fusionar hallazgos, no solo listarlos.

## Pasos para 'Destilar' Info
1. **Identificación de Núcleo:** Extraer el concepto central (X) de cada fuente disponible.
2. **Mapeo Cruzado:** Agrupar las fuentes por subtemas o aspectos del concepto X (ej: Rendimiento, Costo, Arquitectura).
3. **Síntesis Activa:** Para cada grupo temático, sintetizar los puntos comunes y contrastar los divergentes en una única narrativa coherente.
4. **Reducción de Redundancia:** Eliminar frases o datos que se repiten entre fuentes distintas (si el dato es idéntico, solo citarlo una vez con múltiples referencias si es necesario).

**Anti-patrón:** Presentar un resumen por cada fuente individualmente sin intentar entrelazar sus ideas en una visión holística única.
